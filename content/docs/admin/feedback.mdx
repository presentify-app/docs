---
title: Feedback Management
description: Manage topics, change idea statuses, and moderate the Feedback board as a workspace Owner.
---

Workspace **Owners** have additional moderation and organization capabilities on the Feedback board. No separate permission toggle is required — admin features are visible to any user who holds the **Owner** role in at least one workspace.

## Managing topics

Topics are global category tags that users can attach to their ideas (up to 3 per idea).

1. Open the **Feedback** full page
2. Scroll to the **Manage Topics** panel at the bottom (visible only to Owners)
3. Click **Create Topic**, enter a name (max 50 characters), and optionally pick a color with the color picker
4. Use the **pencil** icon to rename a topic or change its color
5. Use the **trash** icon to delete a topic — deleting a topic removes it from all ideas that referenced it

## Changing idea status

1. Open any idea from the Feedback board
2. In the admin panel at the bottom of the idea detail, use the **Change status** dropdown to move the idea through the workflow:

| Status | Meaning |
|---|---|
| **Submitted** | Default state — awaiting review |
| **Under Review** | Actively being evaluated |
| **Planned** | Scheduled for development |
| **Shipped** | Released |

## Deleting ideas and comments

- **Ideas** — In the idea detail view, click **Delete Idea** in the admin panel. This permanently removes the idea, all its votes, and all its comments.
- **Comments** — Owners can delete any comment (not just their own) using the three-dot menu on the comment.

> **Tip**: Post a comment when you change an idea's status — users following the idea will see the context in the comment thread.
