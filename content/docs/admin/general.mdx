---
title: General Settings
description: Configure your workspace name, language, timezone, week start, and authentication methods.
---

General settings apply to your entire workspace and affect all members. Access them via **Settings** > **General**.

## Workspace settings

| Setting | Description | Options |
|---------|-------------|---------|
| **Workspace Name** | Display name shown in the sidebar and workspace selector | Text (max 100 characters) |
| **Default Language** | Language for emails, notifications, and new member defaults | English, Deutsch |
| **Default Timezone** | Timezone used for scheduling, booking displays, and defaults for new members | Any IANA timezone (e.g., Europe/Berlin) |
| **Week Start Day** | First day of the calendar week across the workspace | Monday or Sunday |
| **Allowed Auth Methods** | Which sign-in methods members can use | All, Microsoft only, Magic Link only |

### Authentication methods

- **All** — Members can sign in with Microsoft OAuth or Magic Link email (default)
- **Microsoft** — Only Microsoft 365 sign-in is allowed. Useful for organizations that require corporate credentials.
- **MagicLink** — Only email-based Magic Link sign-in is allowed.

This setting affects the login page — restricted methods are not shown.

## Workspace notifications

- **Weekly Digest** — Toggle to enable or disable the Monday morning weekly digest email for all members. Individual members can also opt out in their profile settings.

## Saving changes

Click **Save** after making changes. Changes apply immediately to the workspace. Language and timezone changes affect new members only — existing members keep their personal settings.
