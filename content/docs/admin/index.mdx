This guide covers everything workspace Admins and Owners need to configure and manage Presentify. If you're setting up a new workspace, start with the [Setup Wizard](/docs/setup).

## Settings overview

Access workspace settings via **Settings** in the sidebar. The settings page has multiple tabs organized into groups:

**Workspace:**
- [General](/docs/admin/general) — Workspace name, language, timezone, auth methods
- [Members](/docs/admin/members) — Invite members, assign roles, manage teams
- [Locations](/docs/admin/locations) — Office locations, buildings, floors
- Holidays — Public holiday calendars

**Booking:**
- [Policies](/docs/admin/policies) — Booking rules, check-in, advance booking limits
- [Resources](/docs/admin/desks) — Desks, parking spots, equipment
- [QR Codes](/docs/admin/qr-codes) — Generate check-in QR codes

**Privacy:**
- [Privacy](/docs/admin/privacy) — Visibility levels, guest booking, data retention

**Billing:**
- [Billing](/docs/admin/billing) — Plans, subscription, usage

**Developer:**
- [API Keys](/docs/integrations/api-keys) — REST API authentication (Enterprise)
- [Webhooks](/docs/integrations/webhooks) — Event notifications (Enterprise)

## Who has access?

Only **Owner** and **Admin** roles can access all settings tabs. Other roles see a limited view:

| Role | Visible Tabs |
|------|-------------|
| Owner / Admin | All tabs |
| Office Admin / Manager / Member | Members (read-only), Billing (read-only) |

See [Roles & Permissions](/docs/roles) for the full permission matrix.
