Die Presentify REST API v1 bietet programmatischen Zugriff auf deine Workspace-Daten. Nutze sie, um eigene Integrationen zu bauen, Workflows zu automatisieren und Daten zu exportieren.

> Die REST API erfordert einen **Enterprise**-Plan und einen [API-Schlüssel](/docs/integrations/api-keys).

## Basis-URL

```
https://app.presentify.org/api/v1
```

## Schnellstart

1. [Erstelle einen API-Schlüssel](/docs/integrations/api-keys) unter **Einstellungen** > **API-Schlüssel**
2. Füge den Schlüssel im `X-API-KEY`-Header bei jeder Anfrage ein
3. Rufe Endpunkte über das tRPC-HTTP-Protokoll auf

```bash
curl -H "X-API-KEY: pk_dein_schluessel" \
  "https://app.presentify.org/api/v1/member.list?input={}"
```

## Verfügbare Router

| Router | Beschreibung |
|--------|-------------|
| [`member`](/docs/api/endpoints#members) | Workspace-Mitglieder auflisten und abrufen |
| [`location`](/docs/api/endpoints#locations) | Bürostandorte auflisten und abrufen |
| [`presence`](/docs/api/endpoints#presence) | Anwesenheitseinträge auflisten, erstellen, aktualisieren und löschen |
| [`booking`](/docs/api/endpoints#bookings) | Buchungen auflisten, abrufen und stornieren |

## Antwortformat

Alle Antworten nutzen das tRPC JSON-RPC Format:

```json
{
  "result": {
    "data": [ ... ]
  }
}
```

Daten verwenden ISO 8601 Format (`2026-03-20T14:30:00Z`). Nur-Datum-Felder nutzen `YYYY-MM-DD`.
