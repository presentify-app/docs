Presentify integriert sich mit Microsoft 365 für Authentifizierung und Profilsynchronisierung und bietet Webhooks sowie eine REST API zur Anbindung externer Systeme.

## Verfügbare Integrationen

<Cards>
  <Card title="Microsoft 365" href="/docs/integrations/entra-id">
    Anmeldung mit Microsoft, Profilsync und Teams SSO.
  </Card>
  <Card title="Kalender-Sync" href="/docs/integrations/calendar-sync">
    Abonniere deinen Presentify-Zeitplan in Outlook, Google oder Apple Kalender.
  </Card>
  <Card title="Microsoft Teams" href="/docs/integrations/teams">
    Teams SSO und automatischer Anwesenheitsstatus-Sync.
  </Card>
  <Card title="Webhooks" href="/docs/integrations/webhooks">
    Erhalte Echtzeit-Benachrichtigungen für Anwesenheit, Buchungen und Mitglieder.
  </Card>
  <Card title="API-Schlüssel" href="/docs/integrations/api-keys">
    Greife auf die Presentify REST API für programmatische Integrationen zu.
  </Card>
</Cards>
