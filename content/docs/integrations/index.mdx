title: Integrations
description: Connect Presentify with Microsoft 365, external systems via webhooks, and the REST API.
---

Presentify integrates with Microsoft 365 for authentication and profile syncing, and provides webhooks and a REST API for connecting with external systems.

## Available integrations

<Cards>
  <Card title="Microsoft 365" href="/docs/integrations/entra-id">
    Sign in with Microsoft, profile sync, and Teams SSO.
  </Card>
  <Card title="Calendar Sync" href="/docs/integrations/calendar-sync">
    Subscribe to your Presentify schedule in Outlook, Google, or Apple Calendar.
  </Card>
  <Card title="Microsoft Teams" href="/docs/integrations/teams">
    Use Presentify within Microsoft Teams with automatic SSO.
  </Card>
  <Card title="Webhooks" href="/docs/integrations/webhooks">
    Receive real-time event notifications for presence, bookings, and members.
  </Card>
  <Card title="API Keys" href="/docs/integrations/api-keys">
    Access the Presentify REST API for programmatic integrations.
  </Card>
</Cards>

## Plan requirements

| Integration | Free Starter | Pro | Enterprise |
|-------------|-------------|-----|------------|
| Microsoft Sign-in | Yes | Yes | Yes |
| Profile Sync | Yes | Yes | Yes |
| Calendar Subscription (iCal) | Yes | Yes | Yes |
| Teams SSO | Yes | Yes | Yes |
| Webhooks | No | No | Yes |
| REST API | No | No | Yes |
