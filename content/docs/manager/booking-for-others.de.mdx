---
title: Für andere buchen
description: Reserviere Schreibtische und Parkplätze im Namen deiner Teammitglieder oder Gäste.
---

Als Manager kannst du Schreibtische und Parkplätze für deine direkten Mitarbeiter buchen. Admins und Owner können für jedes Workspace-Mitglied buchen.

## Wer kann für wen buchen?

| Deine Rolle | Kann buchen für |
|-------------|----------------|
| **Member** | Nur dich selbst |
| **Manager** | Deine direkten Mitarbeiter (Mitglieder, bei denen du als Manager eingestellt bist) |
| **Office Admin** | Mitglieder an deinen zugewiesenen Standorten |
| **Admin** | Jedes Workspace-Mitglied |
| **Owner** | Jedes Workspace-Mitglied |

## Einen Schreibtisch für einen Kollegen buchen

1. Navigiere zu **Schreibtische** und finde einen verfügbaren Schreibtisch
2. Klicke auf den Schreibtisch, um den Buchungsdialog zu öffnen
3. Wähle unter **Buchen für** die Option **Kollege**
4. Suche und wähle das Teammitglied
5. Füge optional Notizen hinzu oder schalte Privatsphäre um
6. Klicke auf **Bestätigen**

Die Buchung wird im Namen des Kollegen erstellt. Er erhält eine Benachrichtigung über die Buchung.

## Einen Schreibtisch für einen Gast buchen

Admins und Manager können für externe Besucher buchen:

1. Öffne den Buchungsdialog für einen Schreibtisch
2. Wähle unter **Buchen für** die Option **Gast**
3. Gib den **Namen** des Gastes ein (erforderlich)
4. Gib optional **E-Mail** und **Firma** ein
5. Klicke auf **Bestätigen**

Gastbuchungen zählen zum täglichen Gastbuchungslimit (Standard: 3 pro Tag pro Standort).

## Delegierte Parkplatzbuchungen

Die gleichen Delegationsregeln gelten für Parkplatzbuchungen. An Standorten mit erweitertem Parkplatzmodus wähle einen Parkplatz und dann "Kollege" im Buchungsdialog.

Im einfachen Parkplatzmodus ist die Parkplatz-Zusatz-Checkbox verfügbar, wenn du einen Schreibtisch für einen Kollegen buchst.

## Delegierte Buchungen stornieren

Du kannst Buchungen, die du für andere erstellt hast, über den **Meine Buchungen**-Tab oder über die **Standort-Buchungen**-Ansicht (Office Admin und höher) stornieren.
