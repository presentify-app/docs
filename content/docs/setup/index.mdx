The Setup Wizard guides new workspace administrators through the initial configuration. It runs automatically after creating a new workspace and walks you through 7 steps.

## Wizard overview

| Step | Name | Required | Description |
|------|------|----------|-------------|
| 1 | [Welcome](#) | — | Introduction and overview |
| 2 | [Workspace Basics](/docs/setup/workspace) | Yes | Name, language, timezone, auth methods |
| 3 | [Locations](/docs/setup/locations) | No | Add office locations |
| 4 | [Desk Setup](/docs/setup/desks) | No | Auto-generate buildings, floors, and desks |
| 5 | [Teams](/docs/setup/teams) | No | Create organizational teams |
| 6 | [Invite Members](/docs/setup/invite) | No | Add members by email or invite link |
| 7 | [Privacy](/docs/setup/privacy) | No | Configure visibility and feature toggles |

After completing (or skipping) all steps, you'll see a success screen with a summary of what was configured.

## Navigation

- **Back** — Return to the previous step
- **Skip** — Skip the current step (available on steps 3–6)
- **Skip Wizard** — Skip all remaining steps and go directly to the dashboard
- **Progress bar** — Shows your completion percentage

## Who can access the wizard?

Only **Owner** and **Admin** roles can access the Setup Wizard. If the setup is already completed, the wizard redirects to the dashboard.

## After the wizard

The wizard sets up the basics. You can fine-tune everything later in [Settings](/docs/admin):

- Add more locations, buildings, and floors
- Configure [booking policies](/docs/admin/policies)
- Set up [zone rules](/docs/admin/zone-rules)
- Upload [floor plans](/docs/admin/floor-plans) (coming soon)
- Create [desk assignments](/docs/admin/desk-assignments)
